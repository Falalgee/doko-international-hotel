# Doko International Hotel Website

A premium, responsive 7‑page hotel website built with HTML, Tailwind CSS, and vanilla JavaScript. Designed for the Doko International Hotel in Minna, Niger State.

## Pages
- Home
- Rooms
- Gallery
- Booking (with Netlify Forms)
- About
- Contact
- Thank You

## Features
- Fully responsive, luxury-inspired design
- WhatsApp booking integration
- Netlify Forms for email submissions
- Image gallery with hover effects
- Testimonial carousel

## Setup
1. Clone the repository
2. Open any HTML file in a browser or serve locally
3. Deploy on Netlify for form handling
